# Laravel Ticketing System

A modern and easy-to-use ticketing web application built with **Laravel**. This system allows users to book tickets, make payments via **M-Pesa**, and validate tickets using **QR codes**. Admins can manage events, monitor bookings, and validate tickets seamlessly.

---

## Features

- **User Features**
  - Browse and book tickets for events.
  - Make payments via M-Pesa.
  - Receive tickets with QR codes.
  - Manual ticket code entry as backup.

- **Admin Features**
  - Manage events and ticket types.
  - Validate tickets using a QR code scanner.
  - View and track all bookings.
  - Manual ticket validation for offline scenarios.

- **Technical Features**
  - Built with Laravel 10.
  - Blade templates for views.
  - TailwindCSS for styling.
  - M-Pesa payment integration.
  - QR code generation and scanning.

---

## Requirements

- PHP >= 8.1
- Composer
- Laravel >= 10
- MySQL or SQLite
- Node.js & npm (for frontend assets)
